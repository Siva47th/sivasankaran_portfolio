Internship Application Automation System

Python Playwright SQLite Streamlit LLM APIs

Project Overview

An end-to-end automation pipeline designed to scrape internship listings across multiple platforms (Internshala, LinkedIn, and Unstop) using Playwright, filtering them against user credentials, generating tailored cover letters, and auto-filling forms to save hundreds of hours of manual entry.

Key Features

  • Multi-Platform Scraping: Playwright scripts to navigate and extract dynamic job card listings.
  • Custom Matching Engine: Scored and filtered listing descriptions against user skills and experience to find optimal matches.
  • Generative AI Integration: Integrates Groq LLM API (Llama 3.1 70B) for instant customized cover letter generation, using Google Gemini API as a backup.
  • Local Storage & Control: Lightweight SQLite database for storage and Streamlit analytics dashboard monitoring the pipelines.
  • Automation & Alerts: Configured APScheduler for automated timed scripts and Google SMTP API for email notification alerts.

Next.js React TailwindCSS Google Genkit Gemini AI

Project Overview

Built a full-stack AI-powered energy monitoring and optimization platform using Next.js 15 (App Router), React 18, and TailwindCSS, with a shadcn/ui and Radix UI component library for a responsive dashboard experience.

Key Features

  • Interactive Dashboard: Developed an interactive single-pane-of-glass energy dashboard integrating real-time consumption monitoring, CSV/JSON historical log uploads, Recharts visualizations, and goal-setting and tracking widgets.
